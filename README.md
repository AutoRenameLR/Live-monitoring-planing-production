# Live Monitoring Planning Production

Aplikasi monitoring produksi yang terhubung langsung dengan data dari Google Sheets secara real-time.

## Fitur Utama
- Menampilkan data produksi terbaru.
- Integrasi otomatis dengan Google Sheets (format CSV).
- Tampilan dashboard yang responsif.

## Cara Penggunaan
1. Update data di Google Sheets yang sudah dipublikasikan.
2. Website akan mengambil data terbaru setiap kali dimuat.
